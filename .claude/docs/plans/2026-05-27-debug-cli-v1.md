# debug-cli v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `debug-cli`, a cross-platform Python CLI that gives AI agents the tools to debug Python code by evidence (run / watch / localize / instrument / interactive debugger session) — like a human at a debugger. JS sibling is v3.

**Architecture:** Lean Python package. Each debug session runs as a background Python subprocess that owns one `debugpy` connection over TCP. The CLI is stateless; it reads `.debug-cli/sessions/<name>/` to find the live session and sends one request → reads one response over a localhost TCP control socket. Multi-session by name. JSON-first output with `--text` for humans.

**Tech Stack:**
- **Python 3.10+** (target also tested on 3.11 and 3.12)
- **`uv`** — env management, lockfile, scripts (`uv init`, `uv add`, `uv run`, `uv sync`)
- **`debugpy`** — the only runtime dep (Microsoft's official Python debugger; same one VS Code uses)
- **Stdlib only otherwise** — `argparse`, `subprocess`, `socket`, `json`, `pathlib`, `ast`, `re`, `signal`, `threading`, `dataclasses`
- **Dev tools** — `pytest`, `pytest-cov`, `ruff` (lint + format), `mypy`
- **Cross-platform** — Windows, macOS, Linux. Process-tree kill abstracted in one helper.

**Design inspirations (wisdom only, no code reuse) from `tmp/debug-skill/`:**
- **Auto-context pattern** — every execution command returns `location + source + locals + stack + recent output` atomically. The single biggest agent-UX win.
- **Multi-session via `--session <name>`** with idle timeout
- **Skill discipline patterns** — "two strikes, rethink"; "stepping 3+ times = needs a breakpoint"; "set breakpoints where the bug *begins*, not where it *manifests*"; wolf-fence loop bisection; deadlock-first concurrency triage
- **Execution-command flag set** — `--break file:line[:cond]` (repeatable), `--remove-break`, `--break-on-exception <filter>`, `--frame N`, `--to file:line` (temp breakpoint), `--stop-on-entry`

**Our differentiators (not in debug-skill):**
1. **Reversible source instrumentation** (`instrument add/list/revert`) — inject logs/breakpoints into source; snapshot original; clean revert
2. **Smart traceback localization** (`localize`) — parse → identify deepest user frame with venv/monorepo path awareness → source context
3. **Regex log/process monitoring** (`watch`) — tail file or run command; match regex; return structured matches
4. **VS Code shared-breakpoints file** (`.debug-cli/breakpoints.json`) as the human↔agent collab medium, plus `--listen <port>` for attach
5. **`diagnose <command>`** — one-call workflow: run crashing thing → parse traceback → start session paused at failure point

| 10 | `session set-bp / clear-bp / list-bp` | Live breakpoint management |
| 11 | `session pause` | Interrupt running program |
| 12 | `session output` | Drain buffered stdout/stderr since last stop |
| 13 | `session restart` | Restart preserving breakpoints |
| 14 | `session release` (alias `session stop`) | Terminate session, clean up |
| 15 | `sessions ls` | List active sessions; auto-clean zombies |
| 16 | `diagnose <command>` | One-call: run → parse traceback → open session at failure |
| 17 | Shared breakpoints file `.debug-cli/breakpoints.json` | Human↔agent collab |

## Auto-Context Format (universal)

Every execution command (`session start/continue/step/restart`) returns this on stop:

```json
{
  "status": "stopped" | "exited" | "error" | "terminated",
  "session_id": "default",
  "reason": "breakpoint" | "step" | "exception" | "pause" | "entry",
  "location": {"file": "src/app.py", "line": 42, "function": "process_items"},
  "source": [
    {"line": 40, "text": "    for item in items:"},
    {"line": 41, "text": "        result = transform(item)"},
    {"line": 42, "text": "        if result is None:", "current": true},
    {"line": 43, "text": "            errors.append(item)"},
    {"line": 44, "text": "            continue"}
  ],
  "locals": [
    {"name": "item", "type": "dict", "value": "{'name': 'test'}", "len": 1},
    {"name": "result", "type": "NoneType", "value": "None"}
  ],
  "stack": [
    {"frame": 0, "function": "process_items", "file": "src/app.py", "line": 42},
    {"frame": 1, "function": "main", "file": "src/app.py", "line": 15}
  ],
  "output": "Processing item: test\n",
  "warnings": []
}
```

**Truncation rules** (mirror debug-skill):
- String values: max 200 chars
- Collections: 5-item preview + total count
- Stack: max 20 frames
- Source: configurable, default 5 lines (2 before, current, 2 after)
- Output: last 200 lines since previous stop

## Conventions Used Throughout the Plan

- **TDD always.** Test first, see it fail, implement, see it pass, commit.
- **One concern per commit.** Smallest meaningful unit.
- **Imports on top.** No mid-file imports (per nir-collab).
- **Type hints everywhere.** `from __future__ import annotations` at top of every module. Use `pathlib.Path`, not strings.
- **Dataclasses for structured data.** No pydantic, no attrs — stdlib only.
- **No mid-file comments explaining what the code does.** Comments are for non-obvious *why*.
- **Ruff for lint + format.** Run after every code change: `uv run ruff check . && uv run ruff format .`
- **Mypy for type check.** Run before commit: `uv run mypy src`.
- **All shell commands timeout explicitly.** Per nir-collab.
- **JSON output default.** `--text` for humans. JSON is single-line (NDJSON-compatible) unless `--pretty`.
- **Errors are structured JSON.** `{"status":"error","error_type":"...","message":"...","details":{...}}` exit code 1.

```python
from __future__ import annotations

import subprocess
import sys
import time
from dataclasses import dataclass
from pathlib import Path


@dataclass
class RunResult:
    exit_code: int
    duration_ms: int
    timed_out: bool
    stdout: str
    stderr: str
    killed_signal: str | None = None


def run_with_timeout(
    cmd: list[str],
    *,
    timeout: float,
    cwd: Path | None = None,
    env: dict[str, str] | None = None,
) -> RunResult:
    start = time.monotonic()
    proc = subprocess.Popen(
        cmd,
        cwd=cwd,
        env=env,
        stdout=subprocess.PIPE,
        stderr=subprocess.PIPE,
        text=True,
        creationflags=_creation_flags(),
    )
    timed_out = False
    try:
        stdout, stderr = proc.communicate(timeout=timeout)
    except subprocess.TimeoutExpired:
        timed_out = True
        _kill_tree(proc.pid)
        stdout, stderr = proc.communicate()
    duration_ms = int((time.monotonic() - start) * 1000)
    return RunResult(
        exit_code=proc.returncode,
        duration_ms=duration_ms,
        timed_out=timed_out,
        stdout=stdout or "",
        stderr=stderr or "",
        killed_signal="SIGTERM" if timed_out else None,
    )


def _creation_flags() -> int:
    if sys.platform == "win32":
        return subprocess.CREATE_NEW_PROCESS_GROUP  # type: ignore[attr-defined]
    return 0


def _kill_tree(pid: int) -> None:
    if sys.platform == "win32":
        subprocess.run(
            ["taskkill", "/F", "/T", "/PID", str(pid)],
            capture_output=True,
            timeout=5,
        )
    else:
        import os
        import signal

        try:
            os.killpg(os.getpgid(pid), signal.SIGTERM)
        except ProcessLookupError:
            pass
```

from __future__ import annotations

import json
from typing import Any


def format_json(data: Any, *, pretty: bool = False) -> str:
    if pretty:
        return json.dumps(data, indent=2, ensure_ascii=False)
    return json.dumps(data, ensure_ascii=False, separators=(",", ":"))


def format_text(data: Any, *, indent: int = 0) -> str:
    pad = "  " * indent
    if isinstance(data, dict):
        lines: list[str] = []
        for k, v in data.items():
            if isinstance(v, (dict, list)):
                lines.append(f"{pad}{k}:")
                lines.append(format_text(v, indent=indent + 1))
            else:
                lines.append(f"{pad}{k}: {v}")
        return "\n".join(lines)
    if isinstance(data, list):
        return "\n".join(format_text(item, indent=indent) for item in data)
    return f"{pad}{data}"
```

import argparse
from dataclasses import asdict

from debug_cli.core.format import format_json, format_text
from debug_cli.core.process import run_with_timeout


def add_subparser(subparsers: argparse._SubParsersAction) -> None:
    p = subparsers.add_parser("run", help="Run a command with timeout, return structured result.")
    p.add_argument("--timeout", type=float, required=True)
    p.add_argument("--cwd")
    p.add_argument("--text", action="store_true", help="Human-readable output instead of JSON.")
    p.add_argument("--pretty", action="store_true", help="Pretty-print JSON.")
    p.add_argument("cmd", nargs=argparse.REMAINDER, help="Command and args after --.")
    p.set_defaults(func=cmd_run)


def cmd_run(args: argparse.Namespace) -> int:
    cmd = args.cmd
    if cmd and cmd[0] == "--":
        cmd = cmd[1:]
    if not cmd:
        print(format_json({"status": "error", "error_type": "usage", "message": "no command provided"}))
        return 1
    result = run_with_timeout(cmd, timeout=args.timeout, cwd=args.cwd)
    payload = asdict(result)
    if args.text:
        print(format_text(payload))
    else:
        print(format_json(payload, pretty=args.pretty))
    return 0

import subprocess
import sys


def scan_process(
    cmd: list[str],
    *,
    patterns: Iterable[str],
    timeout: float,
    until: int | None = None,
    context_lines: int = 1,
) -> Iterator[WatchMatch]:
    compiled = [(p, re.compile(p)) for p in patterns]
    proc = subprocess.Popen(
        cmd,
        stdout=subprocess.PIPE,
        stderr=subprocess.STDOUT,
        text=True,
        bufsize=1,
        start_new_session=(sys.platform != "win32"),
    )
    assert proc.stdout is not None
    deadline = time.monotonic() + timeout
    count = 0
    line_number = 0
    history: list[str] = []
    try:
        for line in proc.stdout:
            line = line.rstrip("\n")
            history.append(line)
            line_number += 1
            for raw_pattern, regex in compiled:
                m = regex.search(line)
                if m:
                    lo = max(0, line_number - 1 - context_lines)
                    hi = line_number + context_lines
                    yield WatchMatch(
                        line_number=line_number,
                        pattern=raw_pattern,
                        match=m.group(0),
                        groups=tuple(m.groups()),
                        surrounding_lines=history[lo:hi],
                        timestamp_ms=int(time.time() * 1000),
                    )
                    count += 1
                    if until and count >= until:
                        return
            if time.monotonic() > deadline:
                return
    finally:
        if proc.poll() is None:
            from debug_cli.core.process import _kill_tree  # type: ignore[attr-defined]

            _kill_tree(proc.pid)
        proc.wait(timeout=5)

from __future__ import annotations

import re
from dataclasses import dataclass, field
from pathlib import Path

_FRAME_RE = re.compile(r'^\s*File "(?P<file>[^"]+)", line (?P<line>\d+), in (?P<func>.+)$')
_ERROR_RE = re.compile(r"^(?P<type>[A-Z]\w*(?:Error|Exception|Warning|Exit|Interrupt))(?::\s*(?P<msg>.*))?$")


@dataclass
class TracebackFrame:
    file: str
    line: int
    func: str
    code: str = ""
    is_user_code: bool = True


@dataclass
class ParsedTraceback:
    error_type: str = ""
    message: str = ""
    frames: list[TracebackFrame] = field(default_factory=list)
    deepest_user_frame: TracebackFrame | None = None
    chained: list[ParsedTraceback] = field(default_factory=list)
    raw: str = ""


def parse_traceback(text: str) -> ParsedTraceback:
    result = ParsedTraceback(raw=text)
    lines = text.splitlines()
    i = 0
    while i < len(lines):
        m = _FRAME_RE.match(lines[i])
        if m:
            code = lines[i + 1].strip() if i + 1 < len(lines) else ""
            result.frames.append(
                TracebackFrame(
                    file=m.group("file"),
                    line=int(m.group("line")),
                    func=m.group("func"),
                    code=code,
                )
            )
            i += 2
            continue
        em = _ERROR_RE.match(lines[i])
        if em:
            result.error_type = em.group("type")
            result.message = em.group("msg") or ""
        i += 1
    result.deepest_user_frame = _find_deepest_user_frame(result.frames)
    return result


def _find_deepest_user_frame(frames: list[TracebackFrame]) -> TracebackFrame | None:
    for frame in reversed(frames):
        path = frame.file
        if any(marker in path for marker in ("site-packages", "/lib/python", "\\Lib\\")):
            frame.is_user_code = False
        else:
            return frame
    return frames[-1] if frames else None
```

Key correctness details:
- Snapshot the file ONCE per file (subsequent adds on the same file don't re-snapshot)
- Insert respects existing indentation of the line below the insertion point
- Insertion line is the line where the user-visible original `line` becomes — i.e., we insert *before* the original line `line`, so all original line numbers below shift by 1
- Track each insertion in `instrumentation.json` with `{id, file, original_line, kind, code, inserted_at_iso}`
- IDs are short (e.g. 8-char hex) — uses `secrets.token_hex(4)`

```python
from __future__ import annotations

import datetime as dt
import secrets
from dataclasses import dataclass
from pathlib import Path

from debug_cli.core.state import ensure_state_dir, read_json, write_json


@dataclass
class Instrumentation:
    id: str
    file: str
    original_line: int
    kind: str
    code: str
    inserted_at: str


def add_instrumentation(file: Path, *, line: int, code: str, kind: str, cwd: Path) -> str:
    state = ensure_state_dir(cwd)
    snapshot = state / "snapshots" / _snapshot_name(file)
    if not snapshot.exists():
        snapshot.write_bytes(file.read_bytes())
    lines = file.read_text(encoding="utf-8").splitlines(keepends=True)
    indent = _detect_indent(lines, line)
    insertion = f"{indent}{code}\n"
    lines.insert(line - 1, insertion)
    file.write_text("".join(lines), encoding="utf-8")
    inst_id = secrets.token_hex(4)
    registry = read_json(state / "instrumentation.json", {})
    assert isinstance(registry, dict)
    registry[inst_id] = {
        "id": inst_id,
        "file": str(file),
        "original_line": line,
        "kind": kind,
        "code": code,
        "inserted_at": dt.datetime.now(dt.timezone.utc).isoformat(),
    }
    write_json(state / "instrumentation.json", registry)
    return inst_id


def list_instrumentations(*, cwd: Path) -> list[Instrumentation]:
    state = ensure_state_dir(cwd)
    registry = read_json(state / "instrumentation.json", {})
    assert isinstance(registry, dict)
    return [Instrumentation(**v) for v in registry.values()]


def revert(inst_id: str | None, *, cwd: Path) -> list[str]:
    state = ensure_state_dir(cwd)
    registry = read_json(state / "instrumentation.json", {})
    assert isinstance(registry, dict)
    if inst_id is None:
        files = {entry["file"] for entry in registry.values()}
    else:
        if inst_id not in registry:
            return []
        files = {registry[inst_id]["file"]}
    reverted: list[str] = []
    for file in files:
        snap = state / "snapshots" / _snapshot_name(Path(file))
        if snap.exists():
            Path(file).write_bytes(snap.read_bytes())
            snap.unlink()
            reverted.append(file)
        for k, v in list(registry.items()):
            if v["file"] == file and (inst_id is None or k == inst_id):
                del registry[k]
    write_json(state / "instrumentation.json", registry)
    return reverted


def _snapshot_name(file: Path) -> str:
    return str(file.resolve()).replace("\\", "_").replace("/", "_").replace(":", "_") + ".bak"


def _detect_indent(lines: list[str], at_line: int) -> str:
    if 0 < at_line <= len(lines):
        target = lines[at_line - 1]
        return target[: len(target) - len(target.lstrip())]
    return ""
```

## Test Strategy

- **Unit tests** (`tests/unit/`) — pure functions only, no debugpy. Fast.
- **Integration tests** (`tests/integration/`, marked `@pytest.mark.integration`) — spawn real debugpy adapter, drive DAP. Slower (~5-10s each).
- **E2e tests** (`tests/e2e/`, marked `@pytest.mark.e2e`) — invoke `python -m debug_cli ...` via subprocess. Slowest.

Run subsets:

```powershell
uv run pytest tests/unit -v
uv run pytest -m "not e2e" -v
uv run pytest -m e2e -v
uv run pytest --cov=debug_cli --cov-report=html
```
